# MDQ Quiz

Obsidian 插件，用于将 Markdown 文件渲染为交互式测验。支持选择题、判断题和填空题，文件扩展名为 `.mdq`。

## 功能

- **选择题**：支持单选（`max=1`）、多选（`max=N`）和不定项（`max=all`）
- **判断题**：点击「正确/错误」按钮作答
- **填空题**：在 `{}` 中填写答案
- **源码模式**：可一键切换查看/编辑原始 Markdown 源码

## 文件格式示例

### 选择题

```markdown
:::select
Q1: 以下哪种物质是化学元素周期表中的惰性气体？(单选)

[A] 氧气
[B] 氮气
[C#] 氦气
[D] 氢气
:::
```

### 判断题

```markdown
:::true-false
Q1: 水的沸点在标准大气压下是 100°C。 [T]
Q2: 太阳从西边升起。 [F]
:::
```

### 填空题

```markdown
:::fill-in
Q1: 水的化学式是 {}。
Q2: 彩虹有 {} 种颜色，红色光波长最 {}。
:::
```

## 开发

本项目尚未发布到 Obsidian 社区插件市场。以下是在本地运行的方法：

### 环境要求

- Node.js >= 18
- pnpm

### 安装与构建

```bash
# 安装依赖
pnpm install

# 开发模式（自动监听文件变化，输出到 test 目录）
pnpm dev

# 生产构建
pnpm build
```

### 本地测试

`test/` 目录是一个可供测试的 Obsidian Vault。开发模式下运行 `pnpm dev` 后：

1. 打开 Obsidian
2. 选择 `test/` 作为 Vault
3. 在 **设置 → 第三方插件** 中启用 `MDQ Quiz` 插件
4. 打开任意 `.mdq` 文件即可看到渲染的测验界面

`test/` 目录已包含示例文件：`选择题.mdq`、`判断题.mdq`、`填空题.mdq`、`样式.mdq`。
